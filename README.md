# 🏦 Bank Loan Performance Analysis (Excel)

This project presents a comprehensive **Bank Loan Performance Analysis** built entirely using **Microsoft Excel**. The report enables stakeholders to monitor lending performance, evaluate loan portfolio health, and identify key trends through KPI-driven dashboards and structured financial analysis.

---

## 🎯 Business Objective

To create an **Excel-based reporting solution** that:
- Tracks key loan performance metrics over time  
- Evaluates portfolio quality through **Good Loan vs Bad Loan** classification  
- Supports data-driven decision-making for lending and risk assessment  

---

## 🛠️ Tools & Skills Used

- Microsoft Excel  
- Pivot Tables & Pivot Charts  
- Advanced Excel Formulas  
- KPI Design & Financial Metrics  
- Data Cleaning & Data Preparation  
- Dashboard Design & Reporting  

---

## 📊 Dashboard 1: Loan Summary Dashboard

The **Summary Dashboard** provides a high-level view of overall lending performance using key KPIs:

- Total Loan Applications  
- Month-to-Date (MTD) and Month-over-Month (MoM) trends  
- Total Funded Amount  
- Total Amount Received  
- Average Interest Rate  
- Average Debt-to-Income Ratio (DTI)  

### Good Loan vs Bad Loan Analysis
Loans are classified using loan status categories:
- **Good Loans:** Fully Paid, Current  
- **Bad Loans:** Charged Off  

Key metrics analyzed include:
- Application count  
- Funded amount  
- Total amount received  

---

## 📈 Dashboard 2: Loan Overview Dashboard

The **Overview Dashboard** focuses on trend analysis and segmentation using Excel-based visualizations:

- **Monthly Trend Analysis** by loan issue date  
- **Regional Analysis** by state  
- **Loan Term Distribution** (e.g., 36 vs 60 months)  
- **Employment Length Analysis**  
- **Loan Purpose Analysis**  
- **Home Ownership Analysis**  

These views help identify lending patterns, borrower behavior, and risk indicators.

---

## 📂 Supporting Data

The project includes a structured loan-level dataset used for:
- KPI calculations
